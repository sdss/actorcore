
actorcore's documentation
=========================

This page documents the message passing system used by `SDSS <https://sdss.org>`__ at Apache Point Observatory and Las Campanas Observatory. It covers the products `tron <https://github.com/sdss/tron>`__, `opscore <https://github.com/sdss/opscore>`__, `actorkeys <https://github.com/sdss/actorkeys>`__, and `actorcore <https://github.com/sdss/actorcore>`__. The documentation for `twistedActor <https://github.com/ApachePointObservatory/twistedActor>`__ and `CLU <https://clu.readthedocs.org>`__ can be found separately.

The current version of actorcore is |actorcore_version|.
